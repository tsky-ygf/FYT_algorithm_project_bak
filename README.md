- DocumentReview 智能文档审核模块
- IntelligentConsultation 智能咨询模块
- LawEntityExtraction 法律实体抽取模块
- LawsuitPrejudgment 法律诉讼预判模块
- LegalKnowledge 普法常识模块
- ProfessionalSearch 智能专业检索模块
- RelevantLaws 相关法条模块
- SimilarCaseRetrieval 相似案例模块
- Tools 训练工具模块
- Utils 工具模块
